Learning Generalizability from Randomness

Zero-shot LiDAR semantic segmentation that transfers to any unseen sensor by disentangling scene–sensor geometry.

Zikun Xu, Shuocheng Yang, Jiaohao Wang, Keqiang Li, Jianqiang Wang, Shaobing Xu;
School of Vehicle and Mobility, Tsinghua University
Paper (PDF) arXiv Code Data Video
TL;DR

A single network trained on one LiDAR collapses on a different one. LiSeer treats every LiDAR as a sparse sampler of the same 3D world, generates diverse cross-sensor data from one dataset, and learns what is invariant across sensors — so it segments scans from sensors it has never seen, with no target data.

LiDAR perception is locked into a single-dataset training paradigm: each new or upgraded sensor demands fresh data collection, annotation, and retraining. We reframe the problem geometrically — any LiDAR is simply a sparse sampler of one shared 3D world, so each scan decomposes into scene-intrinsic geometry (SIG), which is sensor-invariant, and sampling geometry (SG), which is sensor-specific. Cross-sensor generalization then reduces to learning SIG while not overfitting SG. LiSeer realizes this with a controllable data engine and two lightweight modules, and is validated on three benchmarks and three real vehicles carrying previously unseen LiDARs.

The idea

Scene–sensor disentanglement

What a scene is should not depend on how a sensor samples it. LiSeer separates the two.

Method

The LiSeer pipeline

A unified pipeline that learns generalizability from randomness: broad exposure to sampling variation, plus guidance toward scene-intrinsic structure.

Exposure

Controllable data engine

Densify each sparse scan, then resample it under randomized, physically grounded sensor configurations — beams, resolution, field of view, and mounting height — turning one dataset into a broad family of cross-sensor observations.

Guidance

S-Film & CBA-CL

S-Film gives sensor variation (SG) an explicit pathway so the shared decoder can focus on SIG; CBA-CL pulls features toward consistent, class-boundary-aware semantics across different samplings of the same scene.
